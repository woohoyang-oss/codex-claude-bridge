# Local Skills

This directory contains repository-local skill examples for making Codex Desktop feel closer to a Claude-style operating environment.

Recommended use:

1. Start with the skills already in this directory.
2. Copy the ones you want into your global Codex skills directory if you want them available across workspaces.
3. Use the template copies under `/Users/wooho/Documents/Playground/templates/codex-desktop-claude-starter/skills` when bootstrapping a new repo.

Suggested first skills:

- `browser-debug`
- `seo-check`
- `ui-audit`
