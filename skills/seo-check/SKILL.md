# SEO Check

Use this skill when the task is to evaluate a page's search-readiness from the browser.

## Workflow

1. Open the target page in a real browser.
2. Read `document.title`, meta description, canonical, robots, Open Graph tags, and language.
3. Check visible heading structure and hero messaging.
4. If relevant, compare the root domain and a key landing page.
5. Summarize:
   - what is present
   - what is missing
   - what matters most to fix first

## Priorities

- title quality
- meta description
- canonical strategy
- Open Graph tags
- visible intent clarity
