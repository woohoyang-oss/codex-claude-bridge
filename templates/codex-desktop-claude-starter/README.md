# Codex Desktop Claude-Style Starter

This starter is the minimum file set that makes a fresh workspace feel closer to a Claude-style coding environment inside Codex Desktop.

It includes:

- `AGENTS.md` for workspace behavior and execution rules
- `.mcp.json.example` for browser and tool wiring
- `CLAUDE_LIKE_WORKFLOW.md` for the human-facing operating loop
- `skills/` examples for recurring tasks

Use it with:

```bash
/Users/wooho/Documents/Playground/setup-codex-claude-workspace.sh /path/to/your/project
```
